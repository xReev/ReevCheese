Configs Hud (In-Game Config Switching)
Toggle Keys Per Config - set a hotkey for each config to swap instantly while playing.
HUD List For Toggled Configs - only configs with a key show up, with their toggle shown.
Active Config Highlight - the HUD marks which config is currently enabled.
Drag To Reposition - move the HUD while the menu is open.
